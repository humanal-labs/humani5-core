# HUMAN I5 — CORE PRINCIPLES  
Signal • Identity • Integrity

## 1. Signal (Sinyal)
Sinyal = Gürültüsüz bilgi + net yön.  
Bir sinyal ancak üç durumda geçerlidir:
- Sessizlikten doğuyorsa  
- Niyet açıksa  
- Gereksiz açıklama yoksa  

### Sinyal Kuralları:
1. “Hız değil; his yön verir.”  
2. “Sessizlik bile yön verir.”  
3. Bilgi → Gürültü oranı yüksek olmalı.  
4. Karar, tek bir anda netleşir (H5-Decision).  

---

## 2. Identity (Kimlik)
Kimlik, sistemin “Ne olduğu” değil, “Nasıl davrandığıdır.”

### Kimlik Kuralları:
1. Tutarlılık = kimlik.  
2. Aynı konuda iki farklı davranış yok.  
3. İnsan x Teknoloji bütünlüğüne sadakat.  
4. Değer → Davranış → Yön → Sonuç zinciri bozulmaz.  

---

## 3. Integrity (Bütünlük)
Bütünlük, sistemin kendisiyle çelişmemesidir.

### Bütünlük Kuralları:
1. Gereksiz sinyal → silinir.  
2. Duygu → düşünce → eylem hizalanır.  
3. Kaçınma noktaları işaretlenir (H5-Blindspot).  
4. Kararların maliyeti hesaplanır.  

---

## 4. Core Architecture (Mimari)
- **core-signal/** → sinyal işleme prensipleri  
- **core-identity/** → kimlik tutarlılığı modelleri
